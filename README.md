# 🔍 WebInspector — Анализатор сайтов

**WebInspector** — это клиентское веб-приложение для быстрого SEO-анализа сайтов. Приложение парсит HTML-код указанного ресурса, собирает метаданные, проверяет структуру и производительность, а затем генерирует понятный отчет с рекомендациями по улучшению.

Проект написан на нативных технологиях (HTML, SCSS, Vanilla JS) без использования тяжелых фреймворков и библиотек.

---

## ✨ Основной функционал

*   **Умный анализ:** Автоматическая валидация URL (добавление `https://`), сбор данных о мета-тегах, заголовках (H1-H3), изображениях, ссылках и замер времени загрузки.
*   **Генерация рекомендаций:** Автоматическая проверка наличия SSL-сертификата, тегов `viewport`, `title`, `description` и формирование практических советов по SEO.
*   **Визуализация отчета:** Результаты выводятся в виде адаптивных карточек, разделенных на 4 понятные категории:
    *   ❌ **Ошибки (Critical)** — требуют немедленного исправления.
    *   ⚠️ **Предупреждения (Warnings)** — желательно исправить для лучшей оптимизации.
    *   ✅ **Успехи (Success)** — метрики, которые реализованы правильно.
    *   📊 **Статистика (Stats)** — общие числовые показатели сайта.
*   **История проверок:** Сохранение последних 15 проверок в `localStorage`. Возможность мгновенно просмотреть сохраненный результат или очистить историю.
*   **Адаптивный дизайн:** Корректное отображение на любых экранах — от мобильных устройств (375px) до широких десктопов (1920px).
*   **Продуманный UI/UX:** Плавные stagger-анимации появления карточек, наглядный спиннер загрузки, приятные эффекты наведения и фокуса.

## 🚀 Дополнительные возможности

*   🌙 **Темная и Светлая темы:** Премиальный минималистичный дизайн с фиолетовым акцентом. Выбранная тема автоматически сохраняется в `localStorage`.
*   📄 **Экспорт в PDF:** Оптимизированная печать отчета через `window.print()` (скрывает лишние элементы UI, разворачивает скрытые тексты и адаптирует верстку под формат А4).
*   📋 **Копирование отчета:** Кнопка для быстрого копирования структурированного текстового отчета в буфер обмена с помощью Web Clipboard API.
*   ⚡ **Проверка скорости:** Замер времени загрузки HTML-кода сайта через Performance API.
*   🛡️ **Каскадный обход CORS:** Использование надежной цепочки из трех публичных прокси-серверов (`allorigins`, `corsproxy`, `codetabs`). Если один сервер падает или блокируется, автоматически используется следующий.
*   ℹ️ **Справочная система:** Встроенное информационное модальное окно с инструкцией по использованию приложения и расшифровкой цветовых индикаторов.

## 🛠 Технологии

*   **HTML5:** Семантическая и доступная разметка.
*   **SCSS (CSS3):** Кастомные свойства (CSS Variables) для темизации, Flexbox/Grid для построения сеток, миксины для удобной адаптивности.
*   **JavaScript (ES6+):** Асинхронные запросы (`fetch`, `async/await`), парсинг DOM (`DOMParser`), работа с `localStorage` и современными Web APIs.

---

## ⚠️ Примечание по безопасности (CORS)

Некоторые крупные сайты (например, Google, Яндекс, банковские порталы) запрещают встраивание своих страниц через заголовки `X-Frame-Options` или `Content-Security-Policy`. 

В таких случаях визуальный предпросмотр может быть недоступен, однако приложение все равно попытается собрать данные через прокси-серверы. Если прокси также блокируются защитой вроде Cloudflare, приложение автоматически перейдет в **«Ограниченный режим»**, уведомив об этом пользователя.
