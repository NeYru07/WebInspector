🔍 WebInspector — Анализатор сайтов
WebInspector — это клиентское веб-приложение для быстрого SEO-анализа сайтов. Приложение парсит HTML-код указанного сайта, собирает метаданные, проверяет структуру и производительность, а затем генерирует понятный отчет с рекомендациями по улучшению.

Проект написан на нативных технологиях (HTML, SCSS, Vanilla JS) без использования тяжелых фреймворков и библиотек.

✨ Основной функционал
Умный анализ: Валидация URL (автодобавление https://), сбор данных о мета-тегах, заголовках (H1-H3), изображениях, ссылках и времени загрузки.
Генерация рекомендаций: Автоматическая проверка наличия SSL, viewport, title, description и формирование советов по SEO.
Визуализация отчета: Результаты выводятся в виде адаптивных карточек, разделенных на 4 категории:
❌ Ошибки (Critical) — требуют немедленного исправления.
⚠️ Предупреждения (Warnings) — желательно исправить.
✅ Успехи (Success) — то, что сделано хорошо.
📊 Статистика (Stats) — числовые показатели сайта.
История проверок: Сохранение последних 15 проверок в localStorage. Возможность мгновенно просмотреть сохраненный результат или очистить историю.
Адаптивный дизайн: Корректное отображение на экранах от 375px (мобильные) до 1920px (десктопы).
UI/UX: Плавные stagger-анимации появления карточек, спиннер загрузки, эффекты ховера и фокуса.

🚀 Бонусные возможности (Дополнительно реализовано)
🌙 Темная / Светлая тема: Премиальный минималистичный дизайн с фиолетовым акцентом. Тема сохраняется в localStorage.
📄 Экспорт в PDF: Оптимизированная печать отчета через window.print() (скрывает лишние элементы, разворачивает скрытые тексты,适配ируется под А4).
📋 Копирование отчета: Кнопка быстрого копирования структурированного текстового отчета в буфер обмена (Web Clipboard API).
⚡ Проверка скорости: Замер времени загрузки HTML-кода сайта через Performance API.
🛡️ Каскадный обход CORS: Использование цепочки из 3-х публичных прокси-серверов (allorigins, corsproxy, codetabs). Если один падает или блокируется — автоматически используется следующий.
ℹ️ Информационное модальное окно: Встроенная инструкция по использованию приложения и расшифровке цветовых индикаторов.

🛠 Технологии
HTML5: Семантическая разметка.
SCSS (CSS3): Кастомные свойства (CSS Variables) для темизации, Flexbox/Grid для верстки, миксины для адаптивности.
JavaScript (ES6+): Асинхронные запросы (fetch, async/await), работа с DOM (DOMParser), LocalStorage API, Web APIs.

⚠️ Примечание по безопасности (CORS)
Некоторые сайты (например, Google, Яндекс, банки) запрещают встраивание своих страниц в iframe через заголовки X-Frame-Options или Content-Security-Policy. В таких случаях визуальный предпросмотр может быть недоступен, однако приложение все равно попытается собрать данные через прокси. Если и прокси блокируются защитой Cloudflare, приложение перейдет в «Ограниченный режим», уведомив об этом пользователя.
